# Socket-Test with AWS
## Author: Austin Wood
### App Purpose:
* Launch and host an event driven program on amazon web services and connect to it from your local machine.
### App Instructions: 
* Navigate to the correct repository in the terminal
* node console.js
* node app.js
#### Additional Info:
